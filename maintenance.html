ASPA Website Project Structure

Directory layout

aspa-website/
│
├─ index.html               # Landing page (hero only, Enter Site → home.html)
├─ home.html                # Main homepage; cards dynamically populated
├─ news.html                # News page; dynamically renders news items
├─ essays.html              # Essays page; dynamically renders essays
├─ minutes.html             # Meeting Minutes page
├─ transparency.html        # Transparency page
├─ maintenance.html         # Site Maintenance page
│
├─ css/
│   └─ xcss1.css            # Custom CSS (minimalist neutral aesthetic)
│
├─ js/
│   ├─ main.js              # Handles dynamic rendering of JSON content
│   └─ helpers.js           # Optional: utilities for formatting dates, filtering, etc.
│
├─ data/
│   ├─ news.json            # Array of news items (title, date, excerpt, link)
│   ├─ essays.json          # Array of essay items (title, author, excerpt, link)
│   ├─ minutes.json         # Meeting minutes items
│   └─ transparency.json    # Transparency items
│
├─ assets/
│   ├─ logo.png             # ASPA logo
│   ├─ hero-bg.jpg          # Landing page background
│   ├─ images/              # Optional: images for news/essays/etc.
│   └─ docs/                # PDFs, other downloadable content
│
├─ README.md
└─ .gitignore
